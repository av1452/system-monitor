# System Control Center Pro

A real-time system monitoring dashboard inspired by Task Manager.

## Features
- CPU, RAM, Disk monitoring
- Live performance graphs
- Process viewer
- Performance modes (Balanced / Performance / Silent)
- Alert system

## Install (Linux / Raspberry Pi)

```bash
git clone https://github.com/YOUR_USERNAME/system-monitor.git
cd system-monitor
chmod +x install.sh
./install.sh
